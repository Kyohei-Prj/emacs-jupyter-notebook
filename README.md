# emacs-jupyter-notebook

A Jupyter notebook editor for Emacs, providing cell-based editing, kernel integration,
LSP support, and rich output rendering within the Emacs ecosystem.

## Status

[![CI](https://github.com/Kyohei-Prj/emacs-jupyter-notebook/actions/workflows/ci.yml/badge.svg)](https://github.com/Kyohei-Prj/emacs-jupyter-notebook/actions/workflows/ci.yml)

## Requirements

- Emacs 29.1+
- `jupyter.el`
- `lsp-mode`
